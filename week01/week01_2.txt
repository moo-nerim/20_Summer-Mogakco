<Learning Objectives>
⊙ HTML 기본 태그 & 속성 공부하기(2)

- 문서구조
<html> // 가장 큰 범주로 감싸줌
<head>
    <title>HTML_배우기</title>
    <meta charset="utf-8"> // 글씨 깨짐을 방지
</head>
<body> 
본문에 해당되는 내용 작성
</body>
</html>

- <p></p> => p태그 (단락 나누는 태그)
    ex)<p>나는 매우 일찍 인생을 무조건 받아들이기로 결정했다.</p>

- <br> => 줄바꿈
    ex)나는 매우 일찍 인생을 무조건 받아들이기로 결정했다.<br>

- img => 이미지사용
    ex)<img src="img.jpg" width="800"  alt="스누피"> // alt => 이미지 깨졌을 때 알려주는 대채제

<Result of learning>
- p태그, 줄바꿈(br), 이미지 사용법 이해함
- 문서구조를 사용하여 배웠던 기본태그 사용해 html 작성

=> html 안에서는 space를 누른다고 줄바꿈이 따로 되지 않기 때문에 줄바꿈을 하기 위해서는 br를
사용하거나, 한 단락씨 나눌 때는 <p></p> 태그를 사용한다.

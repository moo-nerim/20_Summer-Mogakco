<Learning Objectives>
⊙ HTML 기본 태그 공부하기

- <열린태그></닫힌태그> => 기본 태그 형태

- <h1></h1> => 대제목 작성 태그
    ex) <h1>오늘의 명언</h1>

- <a href=" "> => 링크 주소 가리킴

- <ol></ol> => 순서가 있는 리스트 / 여러 리스트들을 그룹핑해줌 (순서있게)

- <li></li>  => 리스트를 나타냄 / 한줄씩 작성

- <ul></ul> => 순서가 없는 리스트 / 여러 리스트들을 그룹핑해줌 (순서없이)

<Learning Objectives>
⊙ HTML table(표) 익히기

《표 기본 태그》

- <table></table> => 표를 사용하겠다라는 의미(가장 큰 범주)

- <tr></tr> => 표의 하나의 행

- <td></td> => 표 한칸

《표의 구조》

<thead>              // <th></th> => 표 첫줄 목록 글씨 두꺼워짐
</thead>
<tbody>
</tbody>
<tfoot>
</tfoot>

《표의 병합》

- <td rowspan="2">00</td>  => 행 2개 병합

- <td colspan="3">00</td>  => 열 3개 병합


<Result of learning>
- html table 사용법에 대해 학습함
- table을 사용하여 간단한 주소록 html 완성

=> html의 table을 사용하는 법을 좀 더 복습하여 완성도 있는
표를 제작하는 것을 연습해야 할 것 같다.
